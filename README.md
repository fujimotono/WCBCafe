# アプリ名　
 WCBCafe

# 概要　
 架空のカフェのホームページ

# 本番環境
 https://fujimotono.github.io/WCBCafe

# 製作背景（意図）　
 カフェのホームページを製作するにあたって　1、TOPページ　2、NEWSページ　3、MENUページ　4、CONTACTページ　の構成で作成いたしました。
